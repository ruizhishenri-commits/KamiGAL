**KamiGAL** 是一款面向 Android 平台的 Galgame / 视觉小说管理与启动工具，支持 Kirikiroid2（KRKR）、ONScripter、Tyrano、Artemis 等多种引擎的游戏管理和直接启动。

> 本项目基于 [YukiHub](https://github.com/xm486/YukiHub) 开发，采用 **GPL-3.0** 开源协议。
>
> 感谢原项目作者 **xm486** 的出色工作。

---

## 特性

- 🎮 **多引擎支持** — 自动识别并启动 KRKR、ONS、Tyrano、Artemis 等引擎的游戏
- 📚 **智能扫描** — 自动扫描设备中的 Galgame，建立个人游戏库
- 🔍 **元数据匹配** — 自动从 VNDB 获取游戏封面、评分、简介等信息
- 📊 **分类管理** — 支持筛选、搜索、排序，按游玩状态管理游戏
- 📐 **横竖屏自适应** — 竖屏浏览、横屏详情，随心切换
- 🎨 **深色风格** — 沉浸式深色 UI，适合横屏使用
- 📈 **轻量统计** — 实时查看设备在线人数与今日活跃数据
